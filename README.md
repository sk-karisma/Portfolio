# Portfolio

This repository contains a simple and responsive **Resume & CV webpage** created using **HTML** and **CSS**. It is designed to showcase your professional profile, skills, education, experience, and contact details in a clean, modern format.

---

## Features
- Responsive design that works on desktop and mobile devices
- Clean and professional layout suitable for job applications
- Sections for About Me, Skills, Education, Experience, and Contact
- Easy to customize and extend
- Pure HTML & CSS — no dependencies or frameworks required

---

## Files in This Repository
- `resume.html` → Resume page
- `cv.html` → CV page
- `style.css` → Styling for both pages
